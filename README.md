# 0x04. Loops, Conditions and Parsing

This project introduces essential Bash scripting techniques used in automation and DevOps workflows.  
All scripts are written for **Ubuntu 20.04 LTS** and must follow the ALX Bash requirements.

## Learning Objectives

By the end of this project, I should be able to explain:

- How to create SSH key pairs and why they are used
- The difference between `#!/usr/bin/env bash` and `#!/bin/bash`
- How to use `for`, `while`, and `until` loops
- How to use conditional statements: `if`, `else`, `elif`, and `case`
- How to use the `cut` command for text parsing
- File test operators and comparison operators in Bash
- How to write portable and Shellcheck-compliant Bash scripts

## Project Requirements

- All scripts executed on **Ubuntu 20.04 LTS**
- Allowed editors: `vi`, `vim`, `emacs`
- Scripts must end with a new line
- All scripts must be executable
- **No `awk` allowed**
- All scripts must pass **Shellcheck 0.7.0** with no errors
- First line: `#!/usr/bin/env bash`
- Second line: Comment describing the script
