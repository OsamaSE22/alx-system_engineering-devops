# 0x05. Processes and Signals

## Overview
This project covers Linux processes, PIDs, signals, and process management.  
You will write Bash scripts that retrieve process information, send signals, and handle signals inside scripts.

---

## Learning Objectives
By the end of this project, you should be able to explain:

- What a PID is  
- What a process is  
- How to find a process PID  
- How to kill a process  
- What a signal is  
- Which two signals cannot be ignored  

---

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All scripts interpreted on **Ubuntu 20.04 LTS**
- All files must end with a new line
- `README.md` file is mandatory
- All Bash scripts must be executable
- All scripts must pass **Shellcheck 0.7.0** with no errors
- First line of every script: `#!/usr/bin/env bash`
- Second line: a comment describing what the script does
